Episerver Find enterprise search
Episerver Find enterprise search is part of the service, with one index Option per Application Environment. The number of items Option aligns directly with the number of content items in the Master Package, per each index. A set number of languages and text analysis on those languages is included in each Master and Additional Package for the purpose of search. Each language version of a content item counts as one item. Note that search languages and languages Option for the front end website are independent. This means that you can have searchable content in additional languages, however no text analysis on these languages will be performed. You can choose from a list of Supported Languages during the setup and onboarding process.
